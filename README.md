# Sayohat Agentligi Boshqaruv Tizimi

Bu loyiha HTML, CSS va JavaScript yordamida yozilgan sayohat agentligi boshqaruv tizimining statik demo versiyasidir.

## Funksiyalar

- Landing page va sayt taqdimoti
- Xodimlar ro'yxatdan o'tishi va tizimga kirishi
- Mijozlar, paketlar va bronlarni boshqarish paneli
- Ommabop yo'nalishlarni ko'rsatish
- Mahalliy `localStorage` orqali ma'lumotlarni saqlash
- Bron qilish paytida ma'lumotlarni Supabase jadvaliga yuborish

## Ishga tushirish

1. `index.html` faylini brauzerda oching.
2. Xodim sifatida ro'yxatdan o'ting yoki mavjud foydalanuvchi bilan tizimga kiring.
3. Dashboard orqali mijozlar, paketlar va bronlarni boshqaring.

## Foydalanish

- `open-register` tugmasi xodim ro'yxatdan o'tishni ochadi.
- `open-login` tugmasi tizimga kirishni ochadi.
- Dashboard bo'limida mijozlar, paketlar va bronlarni qo'shish, tahrirlash va o'chirish mumkin.

## Fayllar

- `index.html` — asosiy sahifa va foydalanuvchi interfeysi
- `styles.css` — dizayn va tartib
- `script.js` — ilova mantiqi va ma'lumotlarni boshqarish
